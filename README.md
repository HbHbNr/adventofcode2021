# adventofcode2021
Solutions of https://adventofcode.com/2021/

# Requirements
* Python 3 (developed with Python 3.7)
* Flake8 (optional, for linting)
* pytest (optional, for testing)
* mypy (optional, for static type checking)

# Linting

    ./lint.sh

# Testing

    ./pytest.sh

# Static type checking

    ./mypy.sh
